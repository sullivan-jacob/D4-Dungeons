Website Development

This page is meant to help players know what features are under Development and what bugs are being addressed.
Features will be listed here along with their current progress. Completed features will be listed on this page for two weeks after completion.
Known bugs will be listed here, as well as a description of them and how close they are to being removed. Bugs can be sent to Sullie over discord in the bugs channel.

Features

Theme Selection Memory

The goal of the Theme Selection Memory feature is to maintain the selected theme between pages and sessions, instead of turning dark mode on everytime a page is loaded.
The plan for how to achieve this has been established; however, initial test have achieved unexpected results. The current plan is to meet with a Web Development professor to fix the issues.

New Theme UI

The goal with this feature is to allow for more themes than light and dark mode.
Theme suggestions are needed. Suggestions should have the HEX code of the colors and a description of where each color is used.
A new Theme UI is needed inorder to support more themes. This feature will be worked on once the Theme Selection Memory feature has been completed.

Integrated Digital Character Sheet with DM access to current character sheet

The goal of this feature is to give players a digital character sheet that saves data between sessions, as well as provide the DM access to each player's character sheet. List of included features: Full character sheet; spell list, spell slot, Ki point, bardic inspiration, rage uses, conscentration, and condition tracking; spell and feature descriptions;
This is still in concpet and I have to learn some things in order to impliment this, but it is high on the to-do list.

This is a seporate manifestation of the same bug as above. This will be handled differently from the above bug report, but will be handled at the same time.
